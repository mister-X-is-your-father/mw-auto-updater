# 工程 1: 設定 (Config)

## 目的
アップグレード対象のミドルウェアとバージョンを定義する。

## 入力
- なし（人間が編集）

## 処理
1. `config.toml` を編集してターゲットを設定
2. `run.py` で設定を検証

## 出力
- `config.toml` (検証済み)

## 実行方法

```bash
cd 1_config

# 設定を編集
vim config.toml

# 設定を検証
uv run python run.py
```

## チェックリスト

- [ ] 対象ミドルウェアは正しいか
- [ ] 現在バージョンは正確か
- [ ] ターゲットバージョンは妥当か
- [ ] データソースは適切か

検証が完了したら `2_fetch/` に進む
